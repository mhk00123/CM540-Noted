# 運算符優先順序
| 運算符 | 描述 |
| -- | -- |
| `**` | 次方 3**4 = 3 x 3 x 3 x 3 = 81 |
| `*` `/` `%` | 乘、除、取餘數 |
| `+` `-` | 加、减 |
| `<=` `<` `>` `>=` | 小於等於、小於、大於、大於等於 |
| `+` `-` | 加、减 |
| `==` `!=` | 等於、不等於 |
| `is`  `is not` |  |
| `in` `not in` |  |
| `not` `or` `and` | |

>**^** 在實際開發中，如果搞不清楚運算符的優先級，可以使用括號來確保運算的執行順序。
